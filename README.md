# Project 2: Casear Encryption & Decryption

Using this website one can Encrypt or Decrypt texts for Casear Technique.

Formula:
  
  E(i)=P(i)+k
  D(i)=p(i)-k

  here E=Encrypted text
       D=Decrypted text
       k=key
